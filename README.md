# workify

### Build Setup

```bash
// Backend
# install dependencies
npm install

# serve with hot reload at localhost:5000
npm run dev

// Frontend
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve

# build for production
npm run build

```
